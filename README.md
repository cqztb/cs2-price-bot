# CS2 Price Bot

Telegram-бот для отслеживания цен на предметы внутриигрового рынка
Counter-Strike 2 на площадке Steam. Бот ищет предмет, хранит подписки,
периодически проверяет цену и присылает уведомление при достижении порога.

Курсовая работа по образовательной программе «Программные системы и
автоматизация процессов разработки», направление 09.03.04 «Программная
инженерия».

## Возможности
- поиск предмета с выбором износа (качества) и StatTrak;
- подписка на снижение или рост цены до заданного порога;
- периодическая проверка цен и уведомления в Telegram;
- хранение подписок и истории цен в базе данных SQLite.

## Установка и запуск
1. Установить Python 3.10 или новее.
2. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Создать файл `.env` на основе `.env.example` и указать токен бота,
   полученный у @BotFather в Telegram:
   ```
   BOT_TOKEN=ваш_токен
   DB_PATH=price_bot.db
   CHECK_INTERVAL=600
   ```
4. Запустить:
   ```
   python main.py
   ```

## Команды бота
- `/start` — запуск и регистрация;
- `/search` — узнать цену предмета;
- `/track` — создать подписку;
- `/list` — список подписок;
- `/delete <номер>` — удалить подписку;
- `/history` — история цен по подпискам.

## Тесты и покрытие
```
python -m unittest test_unit.py test_integration.py
python coverage_run.py
```

## Структура проекта
- `main.py` — запуск бота и фоновой проверки цен;
- `config.py` — загрузка настроек из `.env`;
- `handlers.py` — обработчики команд и диалоги;
- `steam_client.py` — получение цены из Steam;
- `logic.py` — разбор цены, условие подписки, сборка имени предмета;
- `storage.py` — работа с базой данных SQLite;
- `scheduler.py` — периодическая проверка цен;
- `notifier.py` — отправка уведомлений;
- `test_unit.py`, `test_integration.py` — тесты;
- `coverage_run.py` — измерение покрытия.
